# Triple Espresso

Web_project_coffeeshop_es.
Descripción del proyecto: Estamos usando en este proyecto funcionalidades para emplear una pagina de una cafeterira en la cual pueden hacer reservaciones, para esto hemos usado varias tecnologias como flex box, formularios de envio de informacion usando input, asi como tambien un checkbox para aceptar teminos y condiciones, entre otros codigos interactivos para enlaces y demas.
Planes de mejora del proyecto. Me gustaria que se implementara tambien la condicion de que las redes llevaran a por lo menos las paginas de inicio de sus respectivas webs, esto para implementar adicional algo que vimos y reforzar un punto extra del trabajo ya visto.
